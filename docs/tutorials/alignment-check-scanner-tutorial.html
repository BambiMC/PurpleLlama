Alignment Check Scanner Tutorial

In this tutorial, we'll explore the demo_alignmentcheck.py script from the examples. The Alignment Check Scanner is designed to detect potentially compromised or misaligned agent behavior in conversation traces. This demo shows how to use the scanner to analyze two example traces: one with high-risk input (compromised) and another with low-risk input (not compromised).
